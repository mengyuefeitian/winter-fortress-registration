# 无尽冬日管理助手

> 为《无尽冬日》联盟活动打造的一站式报名管理小程序 —— 堡垒、兵工厂、峡谷会战、国战、官职，五种活动一个入口，告别 Excel 表格和群里刷屏报名。

## 为什么需要它

联盟活动一多，报名管理很快就会失控：谁报了、报满没、谁是替补、谁该踢掉重新分配……全靠人工翻聊天记录、手动数表格，费时又容易出错。

**无尽冬日管理助手** 把这一切搬进小程序：成员自助报名，管理员实时看到报名/满员状态，数据自动统计，权限分级到人。

## 核心功能

- 🏰 **五种活动全覆盖**：堡垒、兵工厂、峡谷会战、国战、官职报名，一个小程序全搞定
- 👥 **分角色管理**：普通用户 / 盟管 / 区管 / 超级管理员，权限逐级细分，各管一摊
- 🗂 **多分区 / 多联盟**：支持任意多个分区，单分区可容纳 12 个联盟，多区管协同管理
- ✅ **在线申请与审核**：想当盟管/区管？在线提交申请，超管一键审核通过
- 📊 **实时报名状态**：满员自动置灰，容量可按活动类型自定义，谁在排队一目了然
- 📈 **数据统计与截图**：管理员随时查看统计数据，一键截图保存到相册
- 🔔 **审核结果通知**：申请结果、报名状态变化，微信订阅消息直接推送到用户
- 🧹 **自动清理**：过期报名与失效时间段定时自动清空，无需人工维护

## 使用指南

### 普通用户

1. 首页选择自己所在分区
2. 选择想报名的活动（堡垒 / 兵工厂 / 峡谷会战 / 国战 / 官职）
3. 填写昵称与位置信息即可完成报名
4. 在【我的】页面随时查看或取消已报名的活动

### 盟管 / 区管 / 超级管理员

- **盟管**：管理所绑定联盟的活动配置与报名数据，查看统计并截图
- **区管**：创建分区、配置联盟与各类活动、审核盟管申请、管理成员
- **超级管理员**：审核区管申请、管理超管名单、查看全局统计、必要时清空数据

权限从普通用户开始逐级申请：想管理联盟就申请盟管，想管理整个分区就申请区管，均需上级审核通过后生效。

## 部署自己的实例

1. 前往 [微信公众平台](https://mp.weixin.qq.com/) 注册小程序并获取 AppID
2. 在微信开发者工具中打开项目，开通云开发，记录环境 ID
3. 把 `project.config.json` 的 `appid`、`miniprogram/app.js` 和 `miniprogram/cloudbaserc.json` 中的 `env` 换成你自己的环境 ID
4. 在云开发控制台按需创建数据集合（`users`、`zones`、`alliances`、`timeSlots`、`registrations` 等，详见代码内注释）
5. 在 `superAdmins` 集合中添加你的手机号，成为第一位超级管理员
6. 右键 `miniprogram/cloudfunctions/` 下每个云函数目录，选择「上传并部署：云端安装依赖」
7. 点击「上传」提交审核，通过后即可发布

> 版本号统一在 `miniprogram/utils/version.js` 中管理，上传新版本前记得更新。
