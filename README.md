# 无尽冬日堡垒分配 - 微信小程序

一个用于联盟活动报名管理的微信小程序。

## 功能特点

- **多角色系统**：普通用户、管理员、审计员、超级管理员
- **分区管理**：支持创建多个分区，每个分区最多12个联盟
- **时间段配置**：支持多个时间段，可添加备注
- **报名管理**：实时显示报名状态，满员自动置灰
- **数据统计**：所有管理员角色都可按联盟查看统计，支持截图保存
- **清空数据**：管理员、审计员、超管可清空报名数据，支持每周重置

## 技术栈

- 微信小程序原生开发
- 微信云开发
- 云数据库

## 部署步骤

### 1. 注册微信小程序

1. 前往 [微信公众平台](https://mp.weixin.qq.com/) 注册小程序账号
2. 获取小程序 AppID

### 2. 开通云开发

1. 在微信开发者工具中打开项目
2. 点击"云开发"按钮
3. 创建云开发环境，记录环境 ID

### 3. 配置项目

1. 打开 `project.config.json`，填写 `appid`
2. 打开 `miniprogram/app.js`，将 `env: 'your-env-id'` 替换为您的云开发环境 ID
3. 打开 `cloudbaserc.json`，将 `env_id` 替换为您的云开发环境 ID

### 4. 初始化数据库

在云开发控制台中创建以下集合：

- `users` - 用户表
- `admins` - 管理员申请表
- `zones` - 分区表
- `alliances` - 联盟配置表
- `timeSlots` - 时间段配置表
- `registrations` - 报名记录表
- `superAdmins` - 超级管理员表

### 5. 添加超级管理员

在云开发数据库的 `superAdmins` 集合中添加记录：

```json
{
  "phone": "您的手机号",
  "createTime": "当前时间"
}
```

### 6. 部署云函数

1. 在微信开发者工具中，右键点击每个云函数目录
2. 选择"上传并部署：云端安装依赖"

### 7. 上传代码

1. 点击"上传"按钮上传代码
2. 在微信公众平台提交审核
3. 审核通过后发布

## 使用说明

### 超级管理员

1. 使用绑定的手机号登录
2. 可以审核管理员申请
3. 可以查看所有分区和联盟的统计数据
4. 可以设置超管手机号
5. 可以清空任意联盟的报名数据

### 管理员

1. 首先申请成为管理员，等待超管审核
2. 可以创建分区（4位数字编号）
3. 可以修改联盟名称
4. 可以配置时间段和备注
5. 可以清空联盟报名数据

### 审计员

1. 由管理员绑定到联盟
2. 可以为时间段添加备注
3. 可以查看联盟报名数据
4. 可以清空绑定联盟的报名数据

### 普通用户

1. 选择分区和联盟
2. 选择时间段报名
3. 填写昵称和位置（车头/车身）
4. 可以查看和取消自己的报名

## 注意事项

1. 请确保云开发环境已正确配置
2. 首次使用请添加超级管理员手机号
3. 图片资源需要自行添加到 `images` 目录
4. 提交审核前请测试所有功能
5. **清空数据操作不可恢复，请谨慎操作**
