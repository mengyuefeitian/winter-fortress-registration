# 版本开发记录

## v1.0.1 (2026-04-22)

### 新增功能
- 版本号集中管理 (`utils/version.js`)
- 盟管可申请升级为区管
- 已审核记录删除功能
- 首页自动刷新用户角色权限

### Bug 修复
- **审核通过后用户无权限**
  - 问题：userId 传递 openid 导致 updateUserRole 失败
  - 解决：查询用户获取真实 _id，校验后才允许审核
  
- **区管申请在审核列表不可见**
  - 问题：数据库权限设置为"仅创建者可读写"
  - 解决：改为"所有人可读写"

- **审核记录混合展示**
  - 问题：区管/盟管审核入口显示所有记录
  - 解决：按 applyType 过滤查询

- **标签文字不居中**
  - 问题：text 元素与 inline-block 标签对齐不一致
  - 解决：全部改用 view + flex 居中

- **布局空间过大**
  - 解决：压缩 padding、margin、字体大小

### 样式优化
- 管理员审核页面改为横向列表布局
- 标签和昵称同行展示，节省空间
- 按钮和标签文字完美居中
